# projetusti-teste

Teste feito para a empresa Projetus TI.

## Teste calculadora

Neste teste foi utilizado a IDE IntelliJ, Flex, Java, Spark.

Criar no projeto "teste" uma calculadora. Ela deverá apresentar a seguinte estrutura:

- Uma tela, utilizando componentes do spark, onde o usuário irá informar 2 números e uma operação. 

- Os parâmetros informados pelo usuário devem chegar na camada cliente (Java ou Groovy), onde o cálculo será realizado.

- Após realizar os cálculos, o resultado deverá ser exibido para o usuário.

Detalhes:

- Operações possíveis: Adição, Subtração, Multiplicação, Divisão, Exponenciação e Radiciação.

- Domínio das operações:

 * Adição, Subtração, Multiplicação e Divisão -> Números Reais

 * Exponenciação e Radiciação -> Números inteiros diferente de zero.

- A aplicação deve rodar no Tomcat do Intelij sem a necessidade de ter que empacotá-la manualmente.

## Anagrama teste

Neste teste foi utilizado a IDE IntelliJ e Java.

Elaborar um programa em Java ou Groovy que calcule a quantidade de classes de anagramas que existem em um texto. As classes também deverão ser exibidas na saída.

Detalhes:

- A entrada do programa será um arquivo texto (.txt) com o nome "texte.txt" (conforme o arquivo em anexo).

- A saída do programa deve ser um arquivo de texto com o nome "anagramas.txt" (conforme o arquivo em anexo).

- Uma classe de anagramas é um conjunto de palavras com as mesmas letras mas por outra ordem. Exemplo:caras, casar e sacar são anagramas, fio e foi formam outra classe.

- Para efeitos deste problema, uma palavra é um conjunto de letras contínuo entre quaisquer outros caracteres e delimitado por espaço (" "), vírgula (","), ponto (".") ou quebra de linha ("\n"). 

- Não deve ter diferenciação entre as letras são maiúsculas ou minúsculas, isto é, Ti é a mesma palavra que ti ou TI. 

- Para simplificar, o texto não irá conter letras acentuadas.
